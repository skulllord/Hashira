1. Make sure Node.js is installed on your system.
2. Place both input1.json and input2.json in this folder.
3. Open a terminal or command prompt.
4. Navigate to the project directory:
   cd c:\Users\priya\Downloads\secret-finder-bigint-fixed
5. Run the script:
   node test-sample.js

The output will print the secret for both test cases.
